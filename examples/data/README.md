# Data

This directory contains sample data files used by other examples across this project.

- `countries.json` — Sample JSON data with country information
- `Financial Sample.csv` / `Financial Sample.xlsx` — Sample financial spreadsheets
- `Office_365_logo.png` / `office-logo-icon.jpg` — Sample images
- `Sample.html` / `Sample.pdf` — Sample documents used by OneNote page creation examples
- `SharePoint User Guide.docx` — Sample Word document
- `reports/` — Additional report sample data
- `templates/` — Template files
