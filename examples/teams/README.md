# Microsoft Teams Examples

Examples for working with Microsoft Teams via the Graph API.

- **Create a team**: [`create_team.py`](./create_team.py)
- **Create a team from an existing group**: [`create_from_group.py`](./create_from_group.py)
- **List all teams**: [`list_all.py`](./list_all.py)
- **List my teams**: [`list_my_teams.py`](./list_my_teams.py)
- **Send a message to a team channel**: [`send_message.py`](./send_message.py)
- **Check user access**: [`does_user_have_access.py`](./does_user_have_access.py)

See https://learn.microsoft.com/en-us/graph/api/resources/team for API documentation.
