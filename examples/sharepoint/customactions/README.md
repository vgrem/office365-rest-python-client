# Custom Actions

Custom actions add custom functionality to SharePoint sites and lists —
inject scripts, add toolbar buttons, or extend the ribbon.

| What | File | Notes |
|------|------|-------|
| **Add script link** | [`add_script_link.py`](./add_script_link.py) | Inject JavaScript on every page (site scope) |
| **Add list action** | [`add_list_action.py`](./add_list_action.py) | Add a toolbar button to a list |
| **List actions** | [`list_actions.py`](./list_actions.py) | Enumerate custom actions (site + list) |
| **Remove action** | [`remove_action.py`](./remove_action.py) | Delete a custom action by ID |
