# Hub Sites & Home Sites

Manage hub sites — register, associate, enumerate, and set as the intranet landing page.

## 🏗️ Hub Sites

| What | File | Notes |
|------|------|-------|
| **List hub sites** | [`list_hub_sites.py`](./list_hub_sites.py) | Enumerate all hub sites |
| **Register hub site** | [`register_hub_site.py`](./register_hub_site.py) | Make a site a hub |
| **Unregister hub site** | [`unregister_hub_site.py`](./unregister_hub_site.py) | Remove hub status from a site |
| **Associate site** | [`associate_site.py`](./associate_site.py) | Join a site to an existing hub |
| **Get connected hubs** | [`get_connected_hubs.py`](./get_connected_hubs.py) | List hubs associated with a hub |

## 🏠 Home Sites

| What | File | Notes |
|------|------|-------|
| **Set home site** | [`set_home_site.py`](./set_home_site.py) | Set as intranet landing page |
| **List home sites** | [`list_home_sites.py`](./list_home_sites.py) | Enumerate all home sites |
